# 案例管理 — 案例卡片详情 内容骨架

> 生成日期：2026-06-28
> 源文档：`docs/功能设计/功能模块全拆解.md#04-真实案例库管理`（CASE-01/04）
> 目标平台：miniprogram（触控）
> 数据来源：`getCard(cardId)` API

---

## 元信息

| 属性 | 值 |
|------|-----|
| 目标平台 | miniprogram |
| 视图名称 | 案例卡片详情（cases-card-detail） |
| 所属页面 | 案例管理（cases） |
| 页面职责 | 展示单张干预案例卡片的完整四段式内容 + 元信息 |
| 入口来源 | 案例详情页 / 主咨询页参考案例 → `card-detail?cardId={id}` |

### Logic-ID 清单

| ID | 类型 | 用途 |
|----|------|------|
| `data` | `CardDetail \| null` | 卡片完整数据 |
| `loading` | `boolean` | 加载中 |
| `error` | `string \| null` | 加载错误 |

---

## 内容区块

### 区块A — 概览区
- **职责**: 展示卡片的核心元数据
- **内容**:
  - 标题
  - 标签行：行为类型、严重程度、场景、家属端大类（如有）
  - 元信息行：适用年龄范围、循证等级
  - 场景描述（如有）
- **data-testid**: `cd-overview`

### 区块B — 四段式内容（核心）
- **职责**: 展示案例卡片的四段结构化干预内容
- **四段结构**:
  1. 即时安全干预动作
  2. 情绪安抚话术
  3. 后续观察指标
  4. 就医判断标准
- **每段**: 段标题（有区分语义）+ Markdown 渲染内容
- **条件**: 某段内容为空时不渲染该段
- **data-testid**: `cd-quartet-card--{accent}`

### 区块C — 注意事项（如有）
- **职责**: 展示使用该干预方案时需注意的事项
- **条件**: `caution_notes` 非空
- **data-testid**: `cd-section`

### 区块D — 禁忌人群（如有）
- **职责**: 展示不适用此干预方案的人群或场景
- **条件**: `contraindications` 非空
- **data-testid**: `cd-section`

### 区块E — 循证实践标签（如有）
- **职责**: 展示案例关联的循证实践标签列表
- **条件**: `ebp_labels.length > 0`
- **data-testid**: `cd-tags`

### 三态
- Loading: 骨架占位 + "加载中…"
- Error: 错误信息 + "重试"按钮
- Normal: 完整卡片内容

### data-testid 节点清单

```
cd-overview, cd-overview__title, cd-overview__tags, cd-overview__meta
cd-quartet-card--immediate, cd-quartet-card--comforting, cd-quartet-card--observation, cd-quartet-card--medical
cd-section, cd-section__title
cd-tags, cd-tags__item
```
